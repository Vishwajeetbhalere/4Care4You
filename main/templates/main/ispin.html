Smart Wheelchair: Enhancing Mobility and Security

The Smart Wheelchair (SW) is a revolutionary mobility solution that combines advanced technology with user-friendly design to enhance the mobility and security of individuals using wheelchairs. Unlike traditional wheelchairs, which are either manually operated or motorized, the Smart Wheelchair incorporates smart features that provide additional mobility and security features.

Key Features

Motor System

The SW features a motor system with a Battery Management System (BMS) that protects the battery from overcharging and over-discharging. The Smart Charging System (SCS) ensures efficient charging and can only be charged with an intelligent Smart Charger (SC).

GPS Tracking

Equipped with a GPS receiver and relay system, the SW allows users to enable or disable GPS tracking. Users can set their normal path (Geo Fencing) and receive remote alarms if the SW crosses the defined boundary.

Controller System

The SW's multi-processor system includes controllers for the charging system, GPS tracker, anti-theft mechanism, and central communication, providing flexibility and efficiency.

Theft Protection

A panic button, gyro sensors, and power supply locking system provide theft protection. If the chair is locked and moved, a local hooter sounds, and remote alerts with the location are sent to the central application.

Camera

A hidden PTZ camera can be activated to send images of the surroundings, enhancing user security.

IoT Multi-mode Communication

The SW's communication system can communicate with central systems using 4G, GPRS, Bluetooth, Wi-Fi, and RF channels, providing flexibility in communication.

Chair Locator

Users can locate the nearest chair using the mobile application and request assistance. Support staff at public places can also use the application to locate and gather chairs for users.

Benefits

Enhanced mobility and security for wheelchair users.

Easy location and retrieval of wheelchairs in public places.

Customizable communication channels based on user requirements.

Remote monitoring and tracking for caregivers and next of kin.

Conclusion

The Smart Wheelchair is a groundbreaking innovation that enhances the mobility and security of wheelchair users. Its advanced features make it a superior choice for individuals seeking a smart mobility solution. Experience the freedom and security of the Smart Wheelchair today!
